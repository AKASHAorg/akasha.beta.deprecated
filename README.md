Install :
- `npm install`
- `npm run flow-typed`
- `npm run rebuild`

Run :
- `npm run dev` (start both main and renderer)

    Or in separate consoles:
    - `npm run hot-server`
    - `npm run start-hot`

// Run the following from the Console tab of your app's DevTools
require('devtron').install()

Package app:
- `npm run package-linux`
